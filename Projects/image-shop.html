At first I managed to complete the HTML layout of the site using divs. There are search box, hero, carousel and the footer. With the help of CSS, a background image was set in the body (the penguin picture). Using JavaScript, this simple website now have functionality. User can change the picture by the left and right arrow of the carousel in the middle of the page. For this, the Translate method was used alongside with the transform property which basically applies a 2D or 3D transformation to an element. However, the aforementioned property was used for moving the picture to the left and right.

Technologies used in this project:

  • HTML
  • CSS
  • JavaScript
