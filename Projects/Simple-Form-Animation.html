The user must enter his username. If the username is correct the webpage background will remain blue, but if it is wrong the background will be turn into red, in this case all usernames are considered correct providing they have more than 7 characters. This is a login form with animation. The HTML file is rather a simple form layout, and most of the job has been done in CSS and JavaScript. The whole project consists of 4 layers namely:

  • field-name
  • field-email
  • field-password
  • field-finished

In each field some icons from fond awesome website has been included. There are arrow, email, key and heart icon. We want to put all these field on top of each other and to do this we must first set the body to:

                    
                        {
                            Position: relative;
                        }

And then set all fields to:

                    
                        {
                            Position: absolute;
                            Top:50%;
                            Left:50%;
                        }

In JavaScript file we make some methods, one of which select the arrows and attaching an event listener to each arrow to check whether the information that the user is providing is valid or not. If the information are valid the user can proceed to the next page, otherwise the page turns into red and shakes. So as you see below the validation check has been failed because the input character is less than 7 character and the background turned into red.
