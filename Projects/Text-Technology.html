Project Description

This is a class project for our course Text Technology in my current M.Sc. in computational linguistics. I'm doing it with two of my classmates. We plan to build a system to ingest XML corpora into a DB system. Any corpus stored in the Database should then be explorable using a small webapp. Our corpus is one of Shakespeare's plays.

Technologies used in this project:

  • HTML
  • XML
  • MySQL
